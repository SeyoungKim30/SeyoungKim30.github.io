📠FERP 설명

《FERP》는 기업에서 필요한 여러가지 정보처리 기능들을 공통 데이터베이스로 관리하는 ERP를 외식산업계에 빠르게 확산되고 있는 ‘키오스크’에 접목시키기 위한 프로젝트입니다.

《FERP》는 ERP의 구성 중에서도 인사, 물류, 분석, 재무에 집중합니다. 고객이 키오스크를 통해 주문하는 순간부터 매장의 운영, 본사에서의 관리까지 하나의 통합된 시스템 안에서 이루어지는 서비스를 제공합니다.

고객은 손쉬운 키오스크 사용을 경험하고 본사에선 실시간으로 매장에 대한 정보 확인 및 체계화된 매장 관리가 가능해집니다. 매장에선 프랜차이즈 환경에 최적화된 재고관리 및 발주 서비스를 이용할 수 있고 매장 관리에 필요한 매출분석, 인사, 회계 등의 기능을 제공받습니다.

Framework

  • SPRING

    Spring은 Java 기반의 웹 프레임워크로, Model-View-Controller(MVC) 아키텍처를 제공하여 웹 애플리케이션을 구축할 수 있게 합니다. 이는 대규모의 엔터프라이즈 자바 개발에 대한 포괄적인 기능을 제공하는 Spring Framework의 일부입니다. Spring MVC는 개발자가 애플리케이션의 비즈니스 로직, 프레젠테이션 로직 및 사용자 인터페이스를 분리하여 코드를 유지, 테스트 및 수정하기 쉽게 만듭니다. 이 프레임워크는 컨트롤러를 사용하여 들어오는 요청을 처리하고, 모델을 사용하여 애플리케이션의 데이터를 표현하며, 뷰를 사용하여 클라이언트에게 응답을 렌더링합니다.

  • myBatis

    MyBatis는 자바 기반의 오픈 소스 ORM 프레임워크로, 데이터베이스와 자바 객체 간의 매핑을 지원합니다. MyBatis는 SQL 쿼리를 XML 파일이나 애노테이션을 이용해 작성하고, 이를 자바 객체와 연결하여 데이터베이스에서 데이터를 조회하거나 수정하는 작업을 쉽게 처리할 수 있도록 도와줍니다.

API & Library

  • AJAX

    AJAX는 Asynchronous JavaScript and XML의 약어로, 비동기적으로 서버와 브라우저 간에 데이터를 주고받을 수 있는 기술입니다. AJAX는 JavaScript를 이용하여 서버에 요청을 보내고, 서버에서는 해당 요청에 대한 응답을 XML, JSON, HTML 등의 형식으로 반환합니다. 이후 JavaScript를 이용하여 해당 데이터를 가공하고 웹 페이지에 동적으로 표시합니다.

  • fetch

    JavaScript 내장 라이브러리 입니다. 비동기 데이터전송 방식으로 서버로부터 데이터를 가져오기 위해 사용합니다. Promise 기반이기 때문에 response가 HTTP error status 여도 반환된 Promise 객체로부터 Response를 수신할 수 있다는 특징이 있습니다.

  • J-query 3.6.0

    jQuery는 HTML 문서를 조작하고, 이벤트 처리 및 애니메이션 효과 등을 쉽게 구현할 수 있는 크로스 플랫폼 자바스크립트 라이브러리입니다. jQuery는 선택자를 이용하여 HTML 요소를 선택하고, 선택한 요소를 쉽게 조작할 수 있도록 다양한 메서드와 기능을 제공합니다.

🎛️ERD

담당기능

💰 계정과목

  • 등록된 계정 검색
  • 새 계정 등록 : 이미 존재하는 계정코드 입력시 등록 불가능
  • 사용여부 변경

💰 전표 수동입력

  • 전표일자와 전표번호를 입력해서 검색 (일자만 입력시 해당일 마지막 전표 검색)
  • 현재 전표의 이전,다음전표가 있을 시 버튼 표시
  • 계정코드 입력 시 계정명 출력
  • 2행 미만으로 행 삭제 불가능

💰 거래내역 조회

  • 일자별 조회 : 일자별로 등록된 전표의 계정과목별 합계금액 출력
  • 건별 조회 : 전표별로 결과 출력

🚚 발주 조회

  • 검색할 조회기간을 선택하면 나머지는 비활성화
  • 일자별 조회시 상품별 수량 조회· 상태 일괄 변경 가능

🚚 발주 정산

  • 정산 상태 변경시 자동으로 전표 생성
  • 검색기간 종료월 기준으로 결제상태 일괄변경

🚚 정산서 조회

  • 품목 구분에 따라 부가세액 자동 계산
  • 정산서 본문 인쇄

🚚 불량 신청 관리

  • 변경을 누르면 [불량신청건 처리]의 내용이 변경되서 노출
  • 조정항목 : 체크를 해제하면 disabled. 기본은 required

🔓임시 비밀번호 발급

  • 일치하는 정보가 있을때, 랜덤생성된 임시비밀번호 메일 발송
